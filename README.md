# DNWE-ARITH
> 浮点数计算工具

## Build Setup

``` bash
# install dependencies
npm install dnwe-arith

import arith from 'dnwe-arith'

arith.sum(array) --加法
arith.exc(array) --减法
arith.mul(array) --乘法
arith.div(array) --除法

#demo
arith.sum([1.01,1.02])

```
